# ROAD DAMAGE AND ACCIDENT RISK ANALYTICS DASHBOARD USING BUSINESS INTELLIGENCE

## Project Overview
This project aims to analyze road damage and accident risk using Business Intelligence techniques. The dashboard is developed using Microsoft Power BI to transform raw data into meaningful insights that support effective decision-making and improve road safety.

---

## Objectives
- Identify high-risk road areas  
- Analyze road damage trends over time  
- Monitor repair and maintenance performance  
- Support data-driven decision-making  

---

## Tools & Technologies
- Microsoft Power BI  
- Excel / CSV  
- DAX (Data Analysis Expressions)  

---

## Features
- Executive OverView  
- Road Damage Analysis 
- Accident Risk Analysis  
- Maintenance Analysis 
- City Level Analysis
- Time Trend Analysis

## Key Insights
- High road damage is observed in major cities  
- Accident risk increases with poor road conditions  
- Maintenance delays negatively impact road safety  

---

## Limitations

- No real-time data integration
